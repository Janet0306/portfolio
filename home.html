Projects

Teach

An E-learning platform built using MERN. Students can add courses to their account and complete them.

For my 4th Year Honours project I decided to learn a completely new stack – the MERN stack. Prior to this point I didn't have much experience in NoSQL databases, and although I had a good level of knowledge of JavaScript, I had not used the React framework.

Teach has a secure login/registration system. Once a user registers they are directed to a profile section of the site and encouraged to fill the section out, once this has been completed they will then be directed to their personalised dashboard. The dashboard is the section of the site where a user can view any topics that they have chosen to complete. The dashboard will also display the user's name and allow then to edit their profile or delete their account. Each topic contains a quiz, factsheet and flashcards and there is a search feature so that users can easily locate a topic.

As well as being educational, the site also has a social component. The web app has a section that allows all users to search and see all students that are on the site. There is also a discussion board where users can post questions and fellow users can post replies. Users profiles also have a section for users to add there social media handles so that other users can connect.

All code can be found on GitHub
